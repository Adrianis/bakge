
Building Bakge
==============

Bakge now has a CMake build. To clean generated files, remove the build/ directory.


Unix commandline
 $ mkdir build
 $ cd build
 $ cmake .. -G "Unix Makefiles"
 $ make


Win32 CMake GUI
 1) Find Bakge root directory, set it as source
 2) Set build directory to <Bakge_root>/build
 3) Click 'Configure' at the bottom left
 4) Click 'Generate' at bottom left
 5) Open ALL_BUILD project file
 6) Right click ALL_BUILD and click 'Build'.
